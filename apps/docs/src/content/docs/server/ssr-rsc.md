## Rendering modes

### File-based React SSR

Render file-based React routes on the server with `createReactRenderer`:

```tsx
import { createReactRenderer } from "@lastshotlabs/snapshot/ssr";

const renderer = createReactRenderer({
  routes: import.meta.glob("./pages/**/*.tsx"),
  layout: "./layouts/default.tsx",
});

// Handle incoming requests
export default {
  async fetch(request: Request) {
    const html = await renderer.render(request);
    return new Response(html, { headers: { "content-type": "text/html" } });
  },
};
```

### Manifest-driven SSR

Render routes from the Snapshot manifest:

```tsx
import { createManifestRenderer } from "@lastshotlabs/snapshot/ssr";
import manifest from "./snapshot.manifest.json";

const renderer = createManifestRenderer({
  manifest,
  apiUrl: "/api",
});
```

### React Server Components

RSC-aware rendering uses the core `renderPage` function with RSC options:

```tsx
import { renderPage } from "@lastshotlabs/snapshot/ssr";
import type { RscOptions, RscManifest } from "@lastshotlabs/snapshot/ssr";

// RSC manifest is generated by snapshotSsr() Vite plugin as rsc-manifest.json
const rscManifest: RscManifest = await import("./rsc-manifest.json");
```

Utilities for RSC detection:

```tsx
import { hasUseClientDirective, hasUseServerDirective, buildComponentId } from "@lastshotlabs/snapshot/ssr";
```

## Route prefetching

Prefetch JS and CSS for upcoming navigations:

```tsx
import { usePrefetchRoute } from "@lastshotlabs/snapshot/ssr";

function NavLink({ href, children }) {
  const prefetch = usePrefetchRoute();
  return (
    <a href={href} onMouseEnter={() => prefetch(href)}>
      {children}
    </a>
  );
}
```

Uses the `prefetch-manifest.json` generated by the `snapshotSsr()` Vite plugin.

## Partial Prerendering (PPR)

PPR renders the static shell at build time and streams dynamic content at request time:

```tsx
import { extractPprShell, createPprCache, renderPprPage } from "@lastshotlabs/snapshot/ssr";

// Create a cache at server startup
const cache = createPprCache();

// Extract shells at build time for each route
const shell = await extractPprShell(<App />);
cache.set("/dashboard", shell);

// At request time, serve the cached shell and stream dynamic content
const response = await renderPprPage({ cache, url: "/dashboard", component: <App /> });
```
